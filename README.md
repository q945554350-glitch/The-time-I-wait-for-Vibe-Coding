# 我等待 Vibe Coding 的时间

一个填满 Vibe Coding 生成间隙的五分钟浏览器肉鸽。

## 玩

直接打开 `index.html`，或在目录中运行：

```bash
python3 -m http.server 8000
```

然后访问 <http://localhost:8000>。

- `WASD` / 方向键：移动
- `空格`：闪身
- 攻击会自动锁定最近的麻烦
- 每次迭代提升，从三个“模型增幅”中选择一个
- 第四分钟结束时，合并冲突小姐会以最终 Boss 身份登场
- 主角是携带发光键盘和电脑包的像素上班族

计时到 `05:00` 后，无论战况如何，游戏都会提醒你回去检查生成结果。
